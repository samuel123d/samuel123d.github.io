## Resumo do Projeto

O website foi construído sobre uma arquitetura leve e responsiva, utilizando o template *Lonely* como base. O foco principal é a experiência do utilizador (UX) e a apresentação clara de portfólios técnicos.

| **Frontend** | HTML5, CSS3, JavaScript |
| **Frameworks** | Bootstrap (Responsividade) |
| **Backend** | PHP (Processamento de Formulários) |
| **Hosting** | GitHub Pages |

## Estrutura de Diretórios

A organização do projeto segue as melhores práticas de desenvolvimento web:

* **assets/**: Recursos estáticos (Estilos CSS, Scripts JS e Imagens).
* **dist/**: Versões de distribuição e ficheiros otimizados.
* **forms/**: Integração de lógica para envio de mensagens de contacto.
* **src/**: Código-fonte original e componentes modulares.
* **index.html**: Ponto de entrada principal da aplicação.
